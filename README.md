# Prep60 – 60 Day Developer Preparation Tracker

> **Minimal productivity tracker for structured coding interview preparation.**

Prep60 is a web application designed to help developers stay consistent while preparing for coding interviews.  
It provides a structured **60-day preparation roadmap** and allows users to track daily progress across:

- Data Structures & Algorithms (DSA)
- Core CS subjects (OOP, DBMS, OS)
- AI / ML learning
- Aptitude preparation
- Technical projects

The tracker provides a clean dashboard to monitor progress and maintain consistency during interview preparation.

## 🚀 Features

- **60-Day Structured Tracker**  
  A predefined roadmap that organizes daily preparation tasks.

- **Daily Task Checklist**  
  Simple checkboxes to mark tasks as completed.

- **DSA Problem Tracker**  
  Monitor progress toward solving **120 DSA problems**.

- **Progress Dashboard**  
  Visual statistics and completion rates to track productivity.

- **Local Storage Persistence**  
  All progress and notes are saved in the browser using Local Storage.

- **Minimal Developer UI**  
  Clean, distraction-free interface designed for productivity.

- **Responsive Design**  
  Works smoothly on desktop, tablet, and mobile devices.
